# Business-Intelligence-Analysis-using-Power-BI

## Project Overview

The AdventureWorks Database, developed by Microsoft, serves as a sample for an online transaction processing (OLTP) database. It caters to the needs of Adventure Works Cycles, a fictional multinational manufacturing company.
By meticulously analyzing data, connecting and shaping it using Power Query, constructing a relational model, and incorporating calculated fields with DAX measures, I has successfully created an interactive dashboard for visualizing its performance metrics
